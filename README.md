## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v20+
- [pnpm](https://pnpm.io/) v10+
- [Expo Go](https://expo.dev/go) on your iPhone, or Xcode for iOS Simulator

### Install dependencies

```bash
pnpm install
```

### Mobile app

```bash
pnpm start
```

Scan the QR code with your iPhone camera to open in Expo Go, or press `i` to open the iOS Simulator.

### API

```bash
cd api
pnpm dev
```

API runs on `http://localhost:3000`. Health check: `GET /health`.
